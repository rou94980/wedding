# Local Agent Handoff Files

This `.agents` folder is a project convention for handoff and continuity.

- `handoff/current-status.md`
  - Current implementation status, confirmed commands, known gaps, and next steps
- `tasks/phase-2-seat-search.md`
  - Recommended second-phase task definition
- `prompts/continue-from-another-machine.md`
  - A ready-to-paste prompt for another Codex instance

`AGENTS.md` is the primary file another Codex should read automatically. The files here provide deeper task context.
